# PowerToys Run

## Keybindings

### Calculadora (**=**)

### Carpeta (**?**)

Programa (**.**)
Para lanzar la ejecucción de un programa

### Shell (**>**)
sirve para ejecutar comandos desde la linea de comandos, por ejemplo ping, telnet, etc

### Comandos del sistema  (**!**)
Sirve para ejecutar comando del sistema, por ejemplo "apagar, recicinar, etc"

### Hora y Fecha (**)**)
Para saber fecha y otros

### Conversor de unidades (**%%**)
Para convertir entre unidades, monedas y otros

### Controlador URI (**//**)
Para abrir principalmente, urls

### Terminal de Windows (**_**)
Para lanzar perfiles especificos de shell

### Navegador de ventanas (**<**)
Para poderse mover entre ventanas
